### Stack:
* Angular v14.2
* Rxjs v7.5
* Typescript v4.8.2
* Tailwind v3.0.2

### Overview:
Layout was built with `tailwind` and angular

### Installation:
1. `git clone https://github.com/HackPoint/m-prest.git`
2. `npm install`

### Run:
`nx run client:serve:development`

### Implemented:
1. List
2. Filter by id, title
3. Add
4. Edit
5. Remove

### Not Implemented:
